Commandes

Voici la liste des commandes utilisables (p pour parenthèses) :
.

  • Commandes requérant un seul argument :
    p, oppose, inverse, carre, racine2, cube, cos, sin
  • Commandes requérant deux arguments :
    diff, quotient, racine, puissance, indice, imagepar
  • Commandes pouvant accepter plus de deux arguments :
    somme, produitx, produit, produit.

Exemples :

  • pour $2+2$, les commandes sont (somme 2 2) ;
  • pour $2×x+1$, les commandes sont (somme (produitx 2 x) 1) ;
  • pour $2\left(x+1\right)$, les commandes sont (produit 2 (p (somme x 1))).

Il faut savoir que :

  • Les parenthèses que vous tapez ne font que délimiter les commandes. Pour ajouter des parenthèses à vos expressions, il faut utiliser la commande p, sauf pour les images par une fonction, car la commande imagepar les inclut.
  • $\frac{1}{x}$ peut être obtenue avec (quotient 1 x), mais aussi avec (inverse x).

Abréviations

Ctrl+B déclenche l’expansion d’une abréviation :

  • (s -> (somme |)
  • (d -> (diff |)
  • (o -> (oppose |)
  • (px -> (produitx |)
  • (p -> (produit |)
  • (p. -> (produit. |)
  • (q -> (quotient |)
  • (i -> (inverse |)
  • (ca -> (carre |)
  • (r2 -> (racine2 |)
  • (r -> (racine |)
  • (ca -> (cube |)
  • (pu -> (puissance |)
  • (in -> (indice |)
  • (im -> (imagepar |)

Fonctionnalités

  • On part avec un certain nombre de points, nombre qui diminue avec le temps.
  • On gagne des points en réussissant à reconstituer les expressions proposées. Les expressions plus difficiles rapportent plus de points.
  • L’écran se colore progressivement en rouge pour repérer les élèves en difficulté d’un seul coup d’œil.
  • Un système de rapport permet d’avoir le bilan d’une partie.
  • Pour les enseignants :
    • Le bilan peut être copié-collé dans un mail, pour étude ultérieure ou le travail est fait à la maison.
    • Vous pouvez fournir des séries, qui seront accessibles à des adresses du genre: http://expressions.club/?test (cliquez pour voir).
      Rassurez-vous, il suffit de taper expressions.club?test dans la barre d’adresse du navigateur.

Vos séries

Si vous voulez utiliser ce site avec vos propres séries d’expressions, vous pouvez me contacter. Il vous suffit de préparer un fichier contenant par exemple :

{
    "title": "Titre de la série",
    "config": {
        "email": "destinataire-rapport@example.com",
        "initial_score": 100,
        "point_duration": 15,
        "display_score": true,
        "display_weakness_indicator": true
    },
    "exprs": [
        {"expr": "(somme 2 2)",
         "points": 1
        },
        {"expr": "(somme (produitx 2 x) 1)",
         "points": 2
        },
        {"expr": "(produit 2 (p (somme x 1)))",
         "points": 2
        }
    ]
}
